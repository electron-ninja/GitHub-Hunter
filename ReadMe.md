## Building

You'll need [Node.js](https://nodejs.org) installed on your computer in order to build this app.

```bash
$ git clone https://github.com/wilsonandusa/GitHub-Hunter.git
$ cd GitHub-Hunter
$ npm install
$ npm start
```
